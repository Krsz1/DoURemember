# Game Service

Servicio de juegos interactivos para estimulación cognitiva.

## Juegos Implementados

### HU 3.2 - Recuerda Quién
Juego de memoria episódica y semántica basado en fotografías familiares.

## Instalación

```bash
npm install
```

## Configuración

Copia `.env.example` a `.env` y configura las variables de entorno.

## Ejecución

```bash
npm run dev  # Desarrollo
npm start    # Producción
```

## Puerto

El servicio corre en el puerto `5003` por defecto.
